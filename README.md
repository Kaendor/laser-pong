# Laser Pong

This is a recreation of the classic Pong game.

## How to play

- Use YI and WR to move the paddles (for now)

## How to run

To play, compile the game with `cargo run --release` and that's it.

- [x] Basic pong game
- [ ] Respawn the ball when a score is made
